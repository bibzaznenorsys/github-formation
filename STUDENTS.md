# Student? Start the Module 2 lab

1. [**Fork**](https://github.com/HansLanda14ib/github-formation/fork) this repository (all branches — not main only)
2. On **your fork**: **Settings** → enable **Issues** and **Actions**
3. **Issues** tab → **New issue** → **Module 2 Lab - Student Class Onboarding**
4. Follow your issue step by step

Direct link (replace `YOUR-USERNAME`):

`https://github.com/YOUR-USERNAME/github-formation/issues/new?template=module-2-lab.md`

Then open [labs/module-2/README.md](labs/module-2/README.md) for clone and test commands.
